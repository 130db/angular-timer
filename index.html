Introduction

Directives in AngularJS is a powerful way of building reusable UI components. This simple project will serve as a sample/reference implementation demonstrating its flexibilities by making it usable across runtime (AngularJS, plain simple JavaScript & jQuery)

Basic Example

This simple directive <timer /> markup code will start the timer with the default option of ticking every 1 millisecond

Timer with interval example

This directive <timer interval="1000" id="timer2"/> markup code will run the timer tick every second

Formatted timer

This directive <timer interval="1000" id="timer3"/> markup code will run the timer tick every second

{{minutes}} minutes, {{seconds}} seconds.

Examples

Using timer directive from Angular

  • index.html
  • Demo
<!DOCTYPE html>
<html>
<head>
<title>Timer Directive used from AngularJS</title>
<script src="angular/angular.min.js"></script>
<script src="app/js/timer.js"></script>
<script>
angular.module('MyApp', ['timer']);
function MyAppController($scope) {
$scope.timerRunning = true;

$scope.startTimer = function (){
$scope.$broadcast('timer-start');
$scope.timerRunning = true;
};

$scope.stopTimer = function (){
$scope.$broadcast('timer-stop');
$scope.timerRunning = false;
};
}
MyAppController.$inject = ['$scope'];
</script>
</head>
<body ng-app="MyApp">
<div ng-controller="MyAppController" style="border: 1px solid gainsboro; text-align: center;">
<h2>Single Timer</h2>
<h3><timer/></h3>
<button ng-click="startTimer()" ng-disabled="timerRunning">Start Timer</button>
<button ng-click="stopTimer()" ng-disabled="!timerRunning">Stop Timer</button>
</div>
<br/>
<div ng-controller="MyAppController" style="border: 1px solid gainsboro; text-align: center;">
<h2>Multiple Timers</h2>
<h3>Timer 1: <timer interval="2000"/></h3>
<h3>Timer 2: <timer>{{minutes}} minutes, {{seconds}} seconds.</timer></h3>
<button ng-click="startTimer()" ng-disabled="timerRunning">Start Timers</button>
<button ng-click="stopTimer()" ng-disabled="!timerRunning">Stop Timers</button>
</div>
</body>
</html>
