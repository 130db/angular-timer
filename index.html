Introduction

AngularJS directive is a powerful way of building reusable components. This simple project will serve as a sample/reference implementation demonstrating the power & flexibilities of AngularJS directives by making it truly usable across runtime libraries (AngularJS, plain old JavaScript, jQuery etc.)

Basic Example

This simple directive <timer id="timer1"/> markup code will start the timer with the default configuration option

Timer with interval example

This directive <timer interval="1000" id="timer2"/> markup code will run the timer tick every second
